Path

Results Path: Informe o nome da pasta na qual os resultados da simulação serão armazenados. Se você desejar manter os resultados de uma simulação, altere o nome desta pasta antes de executar novamente o modelo.

Save Images: Define se serão ou não salvas imagens com o estado do modelo.

Frequency to save images: Informe a frequência com que as imagens serão salvas, ou seja, a cada quantos passos de tempo.

Show observer map: Define se será exibido o observer mostrando a execução do modelo.

Save Logfile: Define se a saída do modelo será salva em arquivo.

Database

dbType: Informe o tipo de base de dados a ser utilizado pelo modelo. É possível utilizar como base de dados um banco de dados Mysql ou Access. Também é possível criar um espaço celular virtual (grade de células). A base de dados pode ser construída utilizando um Sistema de Informações Geográficas (SIG), como, por exemplo, o TerraView.

Mysql

Database: Informe o nome do banco de dados utilizado no modelo.

Layer: Informe o layer do banco a ser utilizado.

Theme: Informe o tema do banco a ser utilizado.

Select: Informe as colunas da tabela de atributos a serem utilizadas pelo modelo.

Host: Informe o host do banco de dados a ser utilizado.

User: Informe o nome do usuário do banco de dados a ser utilizado.

Password: Informe a senha do usuário do banco de dados a ser utilizado.

Access

Access Database file: Selecione o arquivo do banco de dados a ser utilizado.

Layer: Informe o layer do banco a ser utilizado.

Theme: Informe o tema do banco a ser utilizado.

Select: Informe as colunas da tabela de atributos a serem utilizadas pelo modelo.

Virtual

xdim, ydim: Informe a dimensão do espaço celular a ser criado.

Parameters about humans

Use Population Data: Define se o modelo utilizará dados de população reais, carregados do BD ou não.

Population Attribute: Informe o nome do atributo da célula que contém os dados de população, isto é, a coluna do BD onde estão estes dados.

Population Percentage: Informe a porcentagem da população carregada do BD que será utilizada pelo modelo. Deve ser um valor maior que 0 e menor ou igual a 1

Percentage of cells with humans: Informe a porcentagem de células que são ocupadas por humanos (deve ser um número real maior 0 e menor ou igual a 1).

Annual human renewal rate: Informe o percentual anual de pessoas que serão renovadas (deve ser um número real maior ou igual a 0 e menor ou igual a 1).

Daily mobility rate: Informe que percentual da população humana irá visitar outras células (deve ser um número real maior ou igual a 0 e menor ou igual a 1).

Daily mobility rate to public locations: Informe que percentual de humanos, dentro do conjunto diário de humanos que visitam outras células, vai visitar as locações públicas (padarias, farmácias, comércio em geral, áreas de lazer, etc) (deve ser um número real maior ou igual a 0 e menor ou igual a 1).

Percentage of assymptomatic infected humans: Informe a porcentagem de humanos infectados que não apresentam sintomas da dengue (deve ser um número real maoir ou igual a 0 e menor ou igual a 1).

Parameters about vectors

Percentage of cells with vectors: Informe a porcentagem de células que contém mosquitos (deve ser um numero real maior que 0 e menor ou igual a 1).

Probability of migration: Informe a probabilidade de um mosquito voar para uma célula vizinha (deve ser um número real maoir ou igual a 0 e menor ou igual a 1).

Daily survival probability: Informe a probabilidade diária de sobrevivência do mosquito (deve ser um número real maior que 0 e menor que 1).

Maximum proportion of vectors by human: Informe a proporção máxima de mosquitos (fêmeas) por humanos (deve ser um número real maior que 0).

Bite frequency: Informe a taxa média diária de pessoas que o mosquito pica (deve ser um número real maior que 0).

Contamination probabilities

Transmission probability from human to vector: Informe a probabilidade de um humano infectar um mosquito (deve ser um número real maior que 0 e menor ou igual a 1).

Transmission probability from vector to human: Informe a probabilidade de um mosquito infectar um humano (deve ser um número real maior que 0 e menor ou igual a 1).

Parameters about location and time

Percentage of cells with public locations: Informe a porcentagem de células que contém locações públicas em seu interior (deve ser um número real maior ou igual a 0 e menor ou igual a 1).

Total number of days: Informe o número total de dias (deve ser um número inteiro maior que 0).

Parameters for initial conditions

Initial number of infected humans: Informe a quantidade inicial de humanos infectados (deve ser um número inteiro maior que 0).

Initial numbers of recovered humans:Informe a quantidade inicial de humanos recuperados (deve ser um número inteiro maior ou igual a 0).
